## Project Overview

This is an AI Attendance System using MediaPipe Face Detection. It allows for real-time face detection via webcam, student registration by capturing face encodings, automatic attendance marking with timestamps, CSV attendance report generation, and face landmark visualization.

## How to Fork and Run

1. **Forking the Project:**
   - Go to the repository page on GitHub.
   - Click the "Fork" button in the top-right corner to create your own copy.

2. **Clone the Repository:**
   - Copy the forked repository URL.
   - Run the following command in your terminal:
     ```
     git clone <your-forked-repo-url>
     ```

3. **Install Requirements:**
   - Navigate to the project directory:
     ```
     cd attendece
     ```
   - Install dependencies:
     ```
     pip install -r requirements.txt
     ```

4. **Run the Application:**
   - Start the main script:
     ```
     python code.py
     ```
